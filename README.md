# Shopping List Progressive Web App
A simple progressive web app acting as a shopping list for a family. Utilises Google Firebase as a database. Features include:

 - Add items to the shopping list
 - Double-click items to delete them from the shopping list
 - Utilises Google Firebase to save and show items for all users
