![Shopping List PWA](src/img/github-header.png)

A simple shopping list PWA utilising Google Firebase for a database. Features include:

- Add shopping list item
- Click to delete the item including a dialog to confirm action
- Validation for empty input field and unselected item requester
- Utilises Google Firebase as a realtime database

🔗 [View Shopping List PWA](https://aakashpereira.github.io/Shopping-List-PWA/)
